技术栈：
- PHP 8.0 => PHP 8.2
- ThinkPHP 8 => Laravel 11
- 管理后台：Vue3 + TypeScript + ElementPlus UI + TailwindCSS
- 小程序：Vue3 + TypeScript + Uniapp + TailwindCSS
- PC端：Vue3 + Nuxt

## 设计目标

1. 完全兼容likeadmin_php数据库表结构、API接口路由等

2. 无需改动任何一行前端代码/任何一个表结构，即可无缝迁移到Laravel
